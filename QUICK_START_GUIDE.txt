🎮 HOW TO USE THE IMPACT SIMULATOR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: SELECT ASTEROID
  • Choose from dropdown menu
  • Real NASA-tracked objects
  • Shows size and hazard status

Step 2: CHOOSE LOCATION
  Option A: Quick City Buttons
    • New York, London, Tokyo, etc.
    • Instant coordinates
    
  Option B: Manual Entry
    • Enter latitude (-90 to 90)
    • Enter longitude (-180 to 180)

Step 3: SIMULATE
  • Click "Simulate Impact"
  • Wait 5-8 seconds
  • View detailed results

Step 4: ANALYZE RESULTS
  • Review all impact metrics
  • Understand consequences
  • Compare scenarios

📊 API ENDPOINTS (For Developers)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

GET /api/neo/recent
  Returns: List of recent Near Earth Objects

GET /api/statistics
  Returns: Aggregated NEO statistics

POST /api/impact/simulate
  Input: {asteroid_id, latitude, longitude}
  Returns: Complete impact assessment

GET /api/visualizations/threat-map
  Returns: Plotly JSON for global map

GET /api/visualizations/threat-analysis
  Returns: Plotly JSON for analytics

  → Check internet connection
  → Refresh the page
  → Check browser console (F12)

Simulation fails:
  → Verify asteroid is selected
  → Check coordinates are valid
  → Ensure USGS API is accessible
  → Try different asteroid/location

Charts not rendering:
  → Wait for page to fully load
  → Check JavaScript console
  → Ensure Plotly.js loaded
